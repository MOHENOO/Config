* Getting Started

** Install

#+BEGIN_SRC shell
git clone -b develop --depth 1 https://github.com/hlissner/doom-emacs ~/.emacs.d
~/.emacs.d/bin/doom install
~/.emacs.d/bin/doom doctor
M-x all-the-icons-install-fonts
#+END_SRC
